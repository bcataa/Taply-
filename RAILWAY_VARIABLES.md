# Variabile de pus în Railway → Variables

Copiază **Name** și **Value** (valoarea o iei din `.env` de pe calculator).

| Name | Value (din .env) |
|------|------------------|
| `SUPABASE_URL` | din .env – SUPABASE_URL |
| `SUPABASE_SERVICE_ROLE_KEY` | din .env – SUPABASE_SERVICE_ROLE_KEY |
| `SUPABASE_JWT_SECRET` | din .env – SUPABASE_JWT_SECRET |
| `STRIPE_SECRET_KEY` | din .env – STRIPE_SECRET_KEY |
| `STRIPE_PRICE_ID_PREMIUM` | din .env – STRIPE_PRICE_ID_PREMIUM |
| `STRIPE_WEBHOOK_SECRET` | din .env – STRIPE_WEBHOOK_SECRET |
| `BASE_URL` | din .env – BASE_URL (ex. https://www.slebb.com) |
| `SUBDOMAIN_DOMAIN` | din .env dacă ai (ex. slebb.com) |

**Cum faci:** Railway → proiect Taply → **Variables** → Add variable → pui Name exact ca în tabel, Value = valoarea din fișierul `.env` de la tine. La final **Redeploy**.
